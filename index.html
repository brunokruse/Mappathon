Processing Example

  1. Download the Syphon Library for Processing

  2. Install as you would any other Processing library. In the Processing->Libraries->Syphon folder.

  3. Run example project in repo.

openFrameworks Example

  1. download the ofxSyphon library from https://github.com/astellato/ofxSyphon
  2. expand the ofxSyphon-master.zip file
  3. open the folder that was created.
  4. delete the “testapps” directory
  5. rename the folder named “example” to “SyphonServerExample”
  6. navigate to your OF installation and find the folder called “apps”.
  7. Copy the “SyphonServerExample” folder to the “myApps” folder inside the “apps” directory
  8. copy the “ofxSyphon-master” folder to the ‘addons’ folder in your OF installation
  9. open the Xcode project in the SyphonServerExample folder
  10. change the ‘scheme’ to ‘SyphonExample’
  11. Expand the files in the Xcode project and select ofxSyphon
  12. control click on the oxfSyphon folder and select ‘Delete’
  13. find the ofxSyphon-master folder you moved earlier in your Finder and drag it into the ‘addons’ folder in Xcode
  14. A dialog box should appear, make sure “Copy items into destination group’s folder” is not checked, and “Create groups for any added folders” is selected and press “Finish”
  15. Follow the directions in this image to copy the syphon framework to the project https://www.dropbox.com/s/n3r388y7v5r8fg8/ofxSyphon-copy-build-phase.png
  16. At last! you’re ready to run the example! Press the big arrow that says “run” and the example should work.
